Cineon

This plugin provides support for the Kodak Cineon image file format. Cineon is a specialized image file format for working with motion picture film.

Supports

  • Image data: RGB 10-bit (the most common variety)
  • Interleaved channels only

File Extensions

  • .cin

References

  • Kodak, "4.5 DRAFT - Image File Format Proposal for Digital Pictures"

Load Options

Color Profile Set the color profile. Options = Auto, None, Film Print. Default = Auto.
Film Print Set the film print color profile values (black point 0-1024, white point 0-1024, gamma). Default = 95, 685, 1.7.
Convert Convert the bit-depth. Options = None, U8, U16. Default = None.

Save Options

Color Profile Set the color profile. Options = Auto, None, Film Print. Default = Film Print.
Film Print Set the film print color profile values (black point 0-1024, white point 0-1024, gamma). Default = 95, 685, 1.7.

DPX

This plugin provides support for the SMPTE Digital Picture Exchange (DPX) image file format. DPX is a specialized image file format for working with motion picture film. DPX is the successor to the Cineon file format with support for additional image and meta data.

Supports

  • Images: 10-bit RGB type "A" packing (the most common variety); 8-bit, 16-bit, Luminance, RGB, RGBA
  • Interleaved channels only

File Extensions

  • .dpx

References

  • SMPTE, "SMPTE 268M-2003", http://www.smpte.org
  • Cinesite, "Conversion of 10-bit Log Film Data To 8-bit Linear or Video Data"

Load Options

Color Profile Set the color profile. Options = Auto, None, Film Print. Default = Auto.
Film Print Set the film print color profile values (black point 0-1024, white point 0-1024, gamma). Default = 95, 685, 1.7.
Convert Convert the bit-depth. Options = None, U8, U16. Default = None.

Save Options

Color Profile Set the color profile. Options = Auto, None, Film Print. Default = Film Print.
Film Print Set the film print color profile values (black point 0-1024, white point 0-1024, gamma). Default = 95, 685, 1.7.
Version Set the file version. Options = 1.0, 2.0. Default = 2.0.
Type Set the file type. Options = Auto, U10. Default = U10.
Endian Set the file endian. Options = Auto, MSB, LSB. Default = MSB.

IFF

This plugin provides support for the Generic Interchange File Format (IFF).

Supports

  • Images: 8-bit, 16-bit, Luminance, Luminance Alpha, RGB, RGBA
  • File compression

References

  • Affine Toolkit (Thomas E. Burge), riff.h and riff.c, http://affine.org
  • Autodesk Maya documentation, "Overview of Maya IFF"

File Extensions

  • .iff
  • .z

Implementation:

  • Mikael Sundell, mikael.sundell@gmail.com

Save Options

Compression >Set the file compression. Options = None, RLE. Default = RLE.

IFL

This plugin provides support for the Autodesk Image File List (IFL) format. IFL is a file format for creating sequences or playlists of other image files. An IFL file simply consists of a list of image file names, one per line.

Supports

  • Read-only

File Extensions

  • .ifl

LUT

This plugin supports two-dimensional lookup table file formats.

Supports

  • Formats: Inferno, Kodak
  • Images: 8-bit, 16-bit, Luminance, Luminance Alpha, RGB, RGBA; 10-bit RGB

File Extensions

  • .lut
  • .1dl

Load Options

Format Set the file format. Options = Auto, Inferno, Kodak. Default = Auto.
Type Set the file bit depth. Options = Auto, U8, U10, U16. Default = Auto.

Save Options

Format Set the file format. Options = Auto, Inferno, Kodak. Default = Auto.

OpenEXR

This plugin provides support for the Industrial Light and Magic OpenEXR image file format.

Requires

  • OpenEXR - http://www.openexr.com

Supports

  • Images: 16-bit float, 32-bit float, Luminance, Luminance Alpha, RGB, RGBA
  • Image layers
  • Display and data windows
  • File compression

File Extensions

  • .exr

Load Options

Color Profile Set the color profile. Options = None, Gamma, Exposure. Default = Gamma.
Gamma Set the gamma color profile value. Default = 2.2.
Exposure Set the exposure color profile values (value, defog, knee low, knee high). Default = 0.0, 0.0, 0.0, 5.0.
Channels Set how channels are grouped. Options = None, Known, All. Default = Known.

Save Options

compression Set the file compression. Options = None, RLE, ZIPS, ZIP, PIZ, PXR24, B44, B44A. Default = None.

QuickTime

File Extensions

  • .mp4
