Command Line

OpenGL Options

-render_filter (minify) (magnify) Set the render filter. Options = nearest, linear, box, triangle, bell, b-spline, lanczos3, cubic, mitchell. Default = linear, nearest.

Image I/O Options

-base (plugin) (option) (default) Set a base option.
-load (plugin) (option) (default) Set a load option.
-save (plugin) (option) (default) Set a save option.

See also: Images

General Options

-time_units (value) Set the time units. Options = timecode, frames. Default = timecode.
-default_speed (value) Set the default speed. Options = 1, 3, 6, 12, 15, 16, 18, 23.98, 24, 25, 29.97, 30, 50, 59.94, 60, 120. Default = 24.
-help, -h Show the help message.
-info Show the information message.
-about Show the about message.
