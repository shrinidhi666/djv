Description

This program provides a command line tool for displaying information about movies and images.

Command Line

Usage

djv_info (image|directory)... [option]...

Options

-x_info, -xi Don't show image information, only file names.
-verbose, -v Show verbose image information.
-file_path, -fp Show file path names.
-seq, -q (value) Set file sequencing. Options = off, sparse, range. Default = range.
-recurse, -r Descend into sub-directories. 
-columns, -c (value)
 Set the number of columns for formatting output. A value of zero disables formatting.

See also: General, Command Line

Examples

 
> djv_info image.sgi image2.sgi

Display image information.

> djv_info image.1-100.sgi

Display image sequence information.

> djv_info ~/pics

Display information about all images within a directory.
