Description

This program provides a command line tool for listing directories with file sequences. File sequences are groups of files that represent frames from a movie or animation. Since there may be a large number of files within a sequence it can be useful to view them as a group instead of individually.

Example Output

el_cerrito_bart.1k.tiff       File 2.23MB darby rw Mon Jun 12 21:21:55 2006
richmond_train.2k.tiff        File 8.86MB darby rw Mon Jun 12 21:21:58 2006
fishpond.1-749.png           Seq 293.17MB darby rw Thu Aug 17 16:47:43 2006

Command Line

Usage

djv_ls [file|directory]... [option]...

Options

-x_info, -xi Don't show information, only file names.
-file_path, -fp Show file path names.
-seq, -q (value) Set file sequencing. Options: off, sparse, range. Default: range.
-recurse, -r Descend into sub-directories.
-hidden Show hidden files.
-filter, -f (value) Set filters (e.g., *, ?).
-columns, -c (value) Set the number of columns used for formatting the output. A value of zero disables formatting.

Sorting Options

-sort, -s (value) Set the sorting. Options: name, type, size, user, permissions, time. Default: name.
-reverse_sort, -rs Reverse the sorting order.
-x_sort_dirs, -xsd Do not sort directories first.

See also: General, Command Line

Examples

 
> djv_ls

List the current directory.

> djv_ls ~/movies ~/pictures

List specific directories.

> djv_ls -sort time -reverse_sort

Sort by time with the most recent first.

> djv_ls -filter "*.tiff" -filter "*.tif"

Show only files ending with .tiff or .tif (note the use of double-quotes).
